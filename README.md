# HIT137 - Group Assignment 2


## Project Overview
This repository contains the programming solutions for HIT137 Group Assignment 2. The assignment consists of two main Python programs:

1. **Text Encryption & Decryption (`q1.py`)**: A script that reads raw text, encrypts it using mathematical shifts based on alphabet halves, and verifies lossless decryption to prevent character collisions.
2. **Mathematical Evaluator (`evaluator.py`)**: A custom recursive descent parser that evaluates mathematical expressions from a text file. It handles precedence, nested parentheses, and unary negation using strictly plain Python functions (no classes).

## Prerequisites
* Python 3.x installed on your system.
* No external libraries are required (only standard built-in modules like `os` and `string`).
